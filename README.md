# OSSMate 🚀

人気の OSS リポジトリを言語ごとに検索できるツールです。
GitHub API を使用して、スター数の多いプロジェクトを一覧表示します。

OSSのトレンドや Good First Issue を、初心者にもわかりやすく視覚化します。  
（Go + React 構成）

---

## 🛠 技術スタック

- フロントエンド: React + TypeScript + Vite
- バックエンド: Go + Gin
- データ取得: GitHub REST API

---

## 🔧 機能
### ✅ 初期リリース（実装済み）
- 言語によるフィルタリング
- スター順のソート表示
- ページングによる表示制御
